# Шифр "Кузнечик" (ГОСТ Р 34.12-2015), ЭЦП (ГОСТ Р 34.10.2012) и хеширование (ГОСТ Р 34.11-2012)

Репозиторий объединяет три проекта:
1) реализацию блочного шифра ГОСТ Р 34.12-2015 "Кузнечик" с несколькими режимами работы (ГОСТ Р 34.13-2015)
2) электронную подпись ГОСТ Р 34.10-2012 и ГОСТ Р 34.11-2012 (хеширование)
3) инструмент для исследования эллиптических кривых над конечными полями
4) инструмент для исследования полей Галуа

## Содержимое

### `gost_34_10_2012_signature.py`

Что реализовано внутри:
- реализация хэш-функции ГОСТ Р 34.11-2012 ("Стрибог")
- преобразования `S`, `P`, `L` и функция сжатия g
- арифметика точек эллиптической кривой
- умножение точки на скаляр
- генерация закрытого и открытого ключа
- формирование и проверка подписи

### `elliptic_curves.py`

Большой исследовательский CLI-модуль по эллиптическим кривым над конечными полями.

Что делает:
- проверяет корректность параметров кривой
- проверяет простоту модуля кривой p тестом Ферма
- перечисляет точки кривой
- реализует операции сложения и умножения точки
- вычисляет порядки точек
- строит подгруппы
- выводит служебную информацию по структуре группы

Какие алгоритмы используются:
- быстрое возведение в степень по модулю
- расширенный алгоритм Евклида
- сложение точек на кривой в аффинных координатах
- перебор точек и анализ порядков элементов группы

### `kuznyechik_cipher.py`

Реализация блочного шифра "Кузнечик" и режимов шифрования поверх него.

Что реализовано:
- нелинейное преобразование `S`
- линейное преобразование `L`
- вспомогательные преобразования `R`, `LSX`, `F`
- генерация раундовых ключей из мастер-ключа
- базовый класс шифра
- режимы `ECB`, `CTR`, `OFB`, `CBC`, `CFB`
- работа с паддингом, IV (синхропосылкой) и файлами через CLI

Какие алгоритмы используются:
- арифметика в поле Галуа `GF(2^8)`
- раундовые преобразования шифра
- поблочная обработка данных
- потоковые и блочные режимы шифрования
