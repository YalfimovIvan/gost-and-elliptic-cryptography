# ГОСТ и эллиптическая криптография

## О репозитории

Репозиторий объединяет три крупных учебных проекта:
- электронную подпись по ГОСТ Р 34.10-2012
- исследование эллиптических кривых над конечными полями
- реализацию блочного шифра "Кузнечик" с несколькими режимами работы

Все алгоритмы реализованы вручную и ориентированы не на промышленное использование, а на изучение внутреннего устройства криптографических примитивов.

## Содержимое

### `gost_34_10_2012_signature.py`

Интерактивная программа для генерации ключей, подписания файла и проверки подписи.

Что реализовано внутри:
- собственная реализация хэш-функции ГОСТ Р 34.11-2012 ("Стрибог")
- преобразования `S`, `P`, `L` и функция сжатия `g_N`
- арифметика точек эллиптической кривой
- умножение точки на скаляр
- генерация закрытого и открытого ключа
- формирование и проверка подписи
- сохранение параметров кривой, ключей и подписи в текстовые файлы

Какие алгоритмы используются:
- модульная арифметика
- расширенный алгоритм Евклида для обратного элемента
- сложение и удвоение точек на эллиптической кривой
- double-and-add для скалярного умножения
- полная цепочка "хэш -> подпись -> верификация"

### `elliptic_curves.py`

Большой исследовательский CLI-модуль по эллиптическим кривым над полями `F_p`.

Что делает:
- проверяет корректность параметров кривой
- проверяет простоту `p` тестом Ферма
- перечисляет точки кривой
- реализует операции сложения и умножения точки
- вычисляет порядки точек
- строит подгруппы
- выводит служебную информацию по структуре группы

Какие алгоритмы используются:
- быстрое возведение в степень по модулю
- расширенный алгоритм Евклида
- сложение точек на кривой в аффинных координатах
- перебор точек и анализ порядков элементов группы

### `kuznyechik_cipher.py`

Реализация блочного шифра "Кузнечик" и режимов шифрования поверх него.

Что реализовано:
- нелинейное преобразование `S`
- линейное преобразование `L`
- вспомогательные преобразования `R`, `LSX`, `F`
- генерация раундовых ключей из мастер-ключа
- базовый класс шифра
- режимы `ECB`, `CTR`, `OFB`, `CBC`, `CFB`
- работа с паддингом, IV и файлами через CLI

Какие алгоритмы используются:
- арифметика в поле Галуа `GF(2^8)`
- раундовые преобразования ГОСТ-шифра
- поблочная обработка данных
- потоковые и блочные режимы шифрования

## Как запускать

```bash
python gost_34_10_2012_signature.py
python elliptic_curves.py
python kuznyechik_cipher.py
```

Все три программы интерактивные и работают через текстовое меню.

## Что важно знать

- `gost_34_10_2012_signature.py` удобен как демонстрация полного жизненного цикла электронной подписи.
- `elliptic_curves.py` полезен для исследования групповой структуры и отладки формул на кривых.
- `kuznyechik_cipher.py` показывает не только ядро шифра, но и практическую обвязку вокруг него: IV, режимы, файлы, hex-ввод.

## Зависимости

Во всех трёх скриптах используется в основном стандартная библиотека Python.
