# GOST and Elliptic Cryptography

This repository groups several cryptography implementations focused on Russian standards, elliptic curves, and finite-field arithmetic.

## Included scripts

- `gost_34_10_2012_signature.py` - digital signature workflow based on GOST R 34.10-2012
- `elliptic_curves.py` - elliptic curve arithmetic and subgroup exploration
- `kuznyechik_cipher.py` - block cipher implementation with multiple operation modes

## Topics

- hashing and signatures
- elliptic curve point arithmetic
- block cipher modes of operation
- educational cryptography tooling

## Run

```bash
python gost_34_10_2012_signature.py
python elliptic_curves.py
python kuznyechik_cipher.py
```
